# Time Management Landing Page

Une landing page moderne pour une application de gestion de temps, développée avec HTML, CSS et JavaScript.

## Aperçu

Cette page web présente les fonctionnalités d'un outil de gestion de temps avec une interface épurée et des animations fluides.

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (GSAP pour les animations)
- GitHub Pages (hébergement)

## Fonctionnalités

- Design responsive adapté à tous les écrans (mobile, tablette, desktop)
- Animations au défilement avec GSAP
- Structure sémantique
- Interface utilisateur moderne

## Visualiser en local

1. Clonez ce dépôt : 
git clone https://github.com/aea002/Test_Software_Landing.git

2. Ouvrez le fichier `index.html` dans votre navigateur.

Aucune installation supplémentaire n'est nécessaire.

Les animations au défilement utilisent la bibliothèque GSAP. Si vous visualisez le site en local, assurez-vous d'avoir une connexion internet pour charger la bibliothèque depuis le CDN.
